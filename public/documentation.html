This web application is designed to offer users access to movie-related information. Users can update their personal details and create a list of their favorite movies.

Business Logic URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Allows a user to be created /users POST None A JSON object of the new user, structured like: {"name": "", "favoriteMovies": []} A JSON object of the new user added, structured like: {"id": "", "name": "", "favoriteMovies": []}
Allows users to change their username /users/:id PUT :id A JSON object with the updated user name, structured like: {"name": ""} A JSON object of the user
Allows users to add a movie to their favorites /users/:id/:movieTitle POST :id, :movieTitle None Text confirming the movie has been added to the user's favorites
Allows users to remove a movie from their favorites /users/:id/:movieTitle DELETE :id, :movieTitle None Text confirming the movie has been removed from the user's favorites
Allows a user to be deleted /users/:id DELETE :id None Text confirming the user has been deleted
Returns a list of all movies /movies GET None None A JSON array of all movies
Returns data about a specific movie by title /movies/:title GET :title None A JSON object of the movie details if found, or an error message if not found
Returns data about movies with a specific genre /movies/genres/:genreName GET :genreName None A JSON array of movies with the specified genre if found, or an error message if not found
Returns data about movies directed by a specific director /movies/directors/:directorName GET :directorName None A JSON object of the director details if found, or an error message if not found
